Desafio 06 de HTML

Desafio das Tags

Esse exercício faz uma lista de coisas que podem ser adicionadas em uma página. O desafio aqui é arrumar uma maneira que, ao passar o mouse sobre um termo, mostre qual é a tag utilizada para fazer isso.

  • Parágrafo
  • Título
  • Imagens
  • Abreviações
  • Trechos de código
  • Listas numeradas
  • Listas com demarcadores
  • Textos subscrito
  • Textos sobrescritos
  • Texto inserido
  • Texto deletado
  • Marca texto
  • Texto em negrito
  • Texto em itálico
